# JKU-STM32-Code
江科大STM32入门教程全套学习代码与实验工程

## 内容说明
本仓库为个人学习江科大STM32教程的完整代码归档，包含所有课时的实验例程、驱动代码和配套工程。

- 基于 STM32F103C8T6 核心板开发
- 使用 Keil MDK5 编译环境
- 代码逐课时编写，注释清晰，便于学习和调试
- 每个实验独立成完整工程，可直接打开编译下载

## 环境配置
1. 安装 Keil uVision5 及 STM32F1xx 系列芯片支持包
2. 安装 ST-Link 仿真器驱动程序
3. 使用ST-Link连接开发板与电脑USB接口
4. 打开对应实验文件夹内的 `.uvprojx` 工程文件即可使用

## 使用方法
1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/CoGre105/JKU-STM32-Code.git
   ```
2. 进入对应课时的实验文件夹
3. 双击打开 `.uvprojx` 工程文件
4. 点击Build按钮编译工程
5. 点击Download按钮将程序下载到开发板

## 注意事项
- 所有代码均在 STM32F103C8T6 核心板上测试通过
- 若使用其他型号STM32单片机，需修改对应的引脚定义和时钟配置
- 编译报错请优先检查Keil版本和STM32F1xx支持包是否安装完整
- 建议配合江科大官方STM32入门视频教程同步学习
